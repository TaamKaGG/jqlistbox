Namespace: options

jqListbox. options

Default options for the jqListbox plugin

Members

staticjqListbox.options.autoSelectOnClickstring boolean

If true then a simple click event handler will be added to your rendered list items.
Default Value:
  • true

staticjqListbox.options.initialEncodedValuesstring boolean

Initial encoded values to populate the listbox from. Use it to set the initial values from encoded (rendered) value without the need of a targetInput.
Default Value:
  • false

staticjqListbox.options.initialValuesArray boolean

Initial values to populate the listbox from. The value must be a JS array.
Default Value:
  • false

staticjqListbox.options.itemSelectorstring

jQuery compatible selector string to define which elements should be treated as list items in the listbox.
Default Value:
  • "li"

staticjqListbox.options.multiselectboolean

If true then multiple selection is enabled in the listbox
Default Value:
  • true

staticjqListbox.options.selectedClassstring boolean

CSS class used to mark selected items in the listbox. This option can be used on most of the cases. For more complex item logic use the itemRenderer and set selectedClass to false.
Default Value:
  • "selected"

staticjqListbox.options.targetInputstring

jQuery compatible selector string to define which element should be used to set the value to. By default the value will be a JSON encoded string.
Default Value:
  • "#listbox-value"
